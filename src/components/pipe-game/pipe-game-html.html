The game is meant to be played with a head start, so pick one you like and click start before you start moving pieces

The goal of the game is to connect the pipes from start to finish

Click on pipes to rotate them

The green line is where you start - line up your start pipe before hitting start

The red line is the exit

Only use the "pick up" buttons if there are empty spaces. after clicking pick up, click the empty cell that you want it to go to
